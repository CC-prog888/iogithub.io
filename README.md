 <h1>咖啡烘焙度解析：淺焙、中焙、深焙差在哪？風味、酸苦與沖煮一次搞懂</h1>

   </div>

 <section class="tldr">
   <h2>文章摘要</h2>
   <p>咖啡烘焙度決定了咖啡的酸苦與醇厚度。<strong>淺焙</strong>保留產地花果酸香（適合手沖）；<strong>中焙</strong>風味平衡百搭；<strong>深焙</strong>帶有焦糖與醇厚苦甜（適合加奶）。烘焙度越深，產地原本的風味就越不明顯。</p>
 </section>
 
 <p class="intro">買咖啡豆時，包裝上的「淺焙、中焙、深焙」是影響風味的關鍵指標。烘焙度不僅決定豆子的顏色，更直接決定了咖啡的酸值、甜感、醇厚度，以及適合的沖煮器具。本文將以最簡明的結構，帶您一次搞懂烘焙度的差異與挑選邏輯。</p>
 
 <h2>一、什麼是咖啡烘焙度？</h2>
 <p>烘焙是生豆透過加熱產生化學反應（水分蒸發、焦糖化、梅納反應）的過程。烘焙時間越長、溫度越高，豆色越深，風味越偏向焦糖與煙燻；烘焙時間越短，則保留越多生豆原有的花果酸香。</p>
 <p>業界常以爆裂聲作為指標：</p>
 <ul>
   <li><strong>一爆：</strong>水分大量逸出，聲音清脆。淺焙多落於此階段。</li>
   <li><strong>二爆：</strong>油脂滲出，聲音細密。深焙通常進入此階段。</li>
 </ul>

 <p class="note">重要觀念：烘焙度會覆蓋產地風味。越淺焙越能喝出產地特色（如衣索比亞的花香），越深焙則產地差異越小。</p>
 
 <h2>三、專業指標：Agtron 數值</h2>
 <p>除了主觀的名詞，業界常使用 <strong>Agtron 數值</strong>（紅外線測量）來客觀定義烘焙度：</p>
 <ul>
   <li><strong>70 以上：</strong>極淺至淺焙</li>
   <li><strong>55–65：</strong>中焙</li>
   <li><strong>45 以下：</strong>中深至深焙</li>
 </ul>
 
 <h2>四、破解 3 大咖啡迷思</h2>
 <ul>
   <li><strong>迷思 1：深焙咖啡因比較高？</strong><br>錯。烘焙會微幅分解咖啡因，同重量下，深焙咖啡因略低於淺焙。影響咖啡因的關鍵在於粉量與萃取方式。</li>
   <li><strong>迷思 2：豆子出油代表新鮮？</strong><br>錯。深焙豆二爆後出油屬正常現象；但若淺／中焙豆出油，通常是放置過久、油脂氧化的警訊。</li>
   <li><strong>迷思 3：淺焙很酸是烘壞了？</strong><br>錯。淺焙的酸是刻意保留的優質水果酸甜。若出現尖銳單薄的酸味，多半是沖煮萃取不足所致。</li>
 </ul>

 <h2>五、不同烘焙度的沖煮建議</h2>
 <ul>
   <li><strong>淺焙（水溫 92–95°C）：</strong>豆質硬，需較高水溫與稍細研磨來釋放風味。</li>
   <li><strong>中焙（水溫 90–92°C）：</strong>容錯率高，水溫適中即可，是新手最不易出錯的選擇。</li>
   <li><strong>深焙（水溫 85–90°C）：</strong>物質易釋出，建議降低水溫以免過度萃取產生焦澀味。</li>
 </ul>

   想品嚐豆子原本的個性選淺焙，追求平衡選中焙，喜歡濃醇奶香則挑深焙。找到最適合自己的風味，就是最好的咖啡。
 </div>
